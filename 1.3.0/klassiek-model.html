Het klassieke model

In het klassieke model wordt regelgeving gecodeerd in een RegelingKlassiek; deze wordt bekendgemaakt met een BesluitKlassiek. Deze sectie gaat in op beide zaken, en licht vervolgens toe hoe een RegelingKlasssiek volgens STOP gewijzigd kan worden.
