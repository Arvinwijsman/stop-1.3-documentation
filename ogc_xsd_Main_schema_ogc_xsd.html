Schema se-filter-imop.xsd

Namespace http://www.opengis.net/ogc
Definitie

Schema voor het samenstellen van een Filter t.b.v. van symbolisatie voor een Geo-Informatieobject in de STOP-standaard.
Informatie

Beschrijving

Dit is een ingeperkte versie van de Filter Encoding standaard. Deze versie bevat alleen dat deel van de standaard dat door STOP ondersteund wordt. Voor toelichting over de toepassing zie SE schema documentatie.
