Element sv:schema

Namespace https://standaarden.overheid.nl/stop/imop/schemata/
Definitie

Internet-adres dat gebruikt kan worden om het schema (als XSD-bestand) op te halen. Het XSD-bestand bevat de formele definities voor de XML-implementatie van het informatiemodel.
